整体说明

  • 为简化目录结构， 约定图文、图片、收藏等id统一为 ‘888888’，同时url上的 ‘888888’ 表示对应模块的id
  • POST交互统一采用postJSON方式发送数据到后台，后台有接收不到参数的情况，
    解决办法：在php中使用如下方法 获取数据
        $postData = file_get_contents('php://input', true);
